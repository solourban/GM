# Legacy public JavaScript archive

These files used to live under `public/`, but they are not loaded by `public/index.html`.

They are kept here as a reference archive so production static hosting only exposes the active app surface. Do not move a file back into `public/` unless it is also added to `public/index.html`, covered by `package.json` `test:public`, and covered by the relevant regression tests.
