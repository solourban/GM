# 입찰가 산정 탭 계산표 요구사항

이 문서는 낙찰노트 v2의 입찰가 산정 탭에서 입력하거나 계산해야 할 항목과 계산 방향을 정리한 초안이다. 실제 구현 전 프론트 표시명, 저장 키, 계산 함수, 테스트 케이스를 함께 확정해야 한다.

## 입력 및 계산 항목

| 항목 | 구분 | 비고 |
| --- | --- | --- |
| 감정가 | 입력/참고 | 사건 기본 정보에서 가져오거나 사용자가 수정 |
| 최저가 | 입력/참고 | 사건 기본 정보에서 가져오거나 사용자가 수정 |
| 입찰가 | 입력 | 보증금과 수익 계산의 기준 |
| 취득세 + 지방세 | 계산/입력 | 세율 또는 직접 입력 방식 검토 |
| 기타 등기비용 | 입력 | 인지, 증지, 등기 관련 부대비용 |
| 법무비 | 입력 | 법무사 비용 등 |
| 미납관리비 | 입력 | 인수 가능 비용 |
| 수리비 | 입력 | 보수 비용 |
| 명도비 | 입력 | 협의금, 이사비 등 |
| 대출금액 | 입력/계산 | 감정가 또는 낙찰가 기준 산출 가능 |
| 대출비율 | 입력 | LTV 가정 |
| 감정가 60% 기준 대출 | 계산 | 감정가 × 60% |
| 낙찰가 80% 기준 대출 | 계산 | 입찰가 × 80% |
| 방공제 금액 | 입력 | 지역과 담보 조건에 따라 달라질 수 있음 |
| 월 이자 | 계산 | 대출금액 × 연이율 / 12 |
| 보유기간 이자 | 계산 | 월 이자 × 보유 개월 |
| 중도상환수수료 | 계산/입력 | 대출금액 × 수수료율 |
| 총 취득비용 | 계산 | 아래 계산식 참고 |
| 필요 현금 | 계산 | 총 취득비용 - 대출금액 |
| 예상 매도가 | 입력 | 출구 가격 가정 |
| 세전수익 | 계산 | 예상 매도가 - 입찰가 |
| 매도 중개수수료 | 입력/계산 | 매도가 기준 |
| 총 비용처리 금액 | 계산 | 취득 관련 비용 + 보유비용 + 매도비용 |
| 과세표준 | 계산 | 세전수익 - 비용처리 금액 |
| 양도세 또는 소득세 | 입력/계산 | 세무 확인 필요 |
| 양도세 지방세 | 입력/계산 | 세무 확인 필요 |
| 총 비용 | 계산 | 총 취득비용 + 매도비용 + 세금 |
| 세후수익 | 계산 | 예상 매도가 - 총 비용 - 세금 |
| 수익률 | 계산 | 세후수익 / 필요 현금 |

## 계산 방향

```txt
총 취득비용 =
입찰가
+ 취득세/지방세
+ 등기비용
+ 법무비
+ 미납관리비
+ 수리비
+ 명도비
+ 보유기간 이자
+ 중도상환수수료

필요 현금 =
총 취득비용 - 대출금액

세전수익 =
예상 매도가 - 입찰가

총 비용처리 금액 =
취득 관련 비용 + 보유비용 + 매도비용

과세표준 =
세전수익 - 비용처리 금액

세후수익 =
예상 매도가 - 총 비용 - 세금

수익률 =
세후수익 / 필요 현금
```

## 주의 문구

- 본 계산은 참고용입니다.
- 실제 대출 가능 여부는 금융기관 확인이 필요합니다.
- 세금은 세무 전문가 또는 관계 기관 확인이 필요합니다.
- 권리관계와 명도 가능성은 원본 서류와 현장 확인이 필요합니다.
