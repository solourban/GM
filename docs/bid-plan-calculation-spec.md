# 입찰가 산정 탭 계산표 요구사항

이 문서는 낙찰노트의 입찰가 산정 탭에서 다뤄야 할 입력값, 참고값, 계산식, 주의 문구를 정리한다. 실제 구현 전후 계산식 변경은 테스트와 함께 검증한다.

## 입력 및 표시 항목

| 항목 | 구분 | 설명 |
| --- | --- | --- |
| 감정가 | 참고값 | 법원 감정평가 기준 금액 |
| 최저가 | 참고값 | 해당 회차 최저매각가격 |
| 입찰가 | 입력값 | 사용자가 계획한 입찰 금액 |
| 취득세 + 지방세 | 입력/계산값 | 취득 관련 세금 |
| 기타 등기비용 | 입력값 | 인지, 증지, 채권 등 등기 관련 비용 |
| 법무비 | 입력값 | 법무사 또는 관련 진행 비용 |
| 미납관리비 | 입력값 | 인수 또는 협의 가능성이 있는 관리비 |
| 수리비 | 입력값 | 보수, 인테리어, 원상회복 예상 비용 |
| 명도비 | 입력값 | 점유 이전 및 협의 관련 예상 비용 |
| 대출금액 | 입력값 | 실제 또는 예상 대출 금액 |
| 대출비율 | 참고값 | 감정가 또는 낙찰가 기준 대출 비율 |
| 감정가 60% 기준 대출 | 계산값 | 감정가 기준 보수적 대출 참고값 |
| 낙찰가 80% 기준 대출 | 계산값 | 낙찰가 기준 대출 참고값 |
| 방공제 금액 | 입력값 | 지역별 소액임차보증금 공제 참고값 |
| 월 이자 | 계산값 | 대출금액과 금리 기준 월 이자 |
| 보유기간 이자 | 계산값 | 월 이자와 예상 보유기간 기준 비용 |
| 중도상환수수료 | 입력/계산값 | 매도 또는 상환 시 발생 가능한 수수료 |
| 총 취득비용 | 계산값 | 입찰부터 보유 이자까지 포함한 취득 총액 |
| 필요 현금 | 계산값 | 총 취득비용에서 대출금액을 뺀 현금 필요액 |
| 예상 매도가 | 입력값 | 매도 시 받을 것으로 예상하는 금액 |
| 세전수익 | 계산값 | 예상 매도가에서 입찰가를 뺀 금액 |
| 매도 중개수수료 | 입력값 | 매도 시 중개 비용 |
| 총 비용처리 금액 | 계산값 | 취득, 보유, 매도 관련 비용 합계 |
| 과세표준 | 계산값 | 세전수익에서 비용처리 금액을 뺀 금액 |
| 양도세 또는 소득세 | 입력/계산값 | 세금 추정액 |
| 양도세 지방세 | 입력/계산값 | 양도세에 연동되는 지방세 |
| 총 비용 | 계산값 | 취득비용, 보유비용, 매도비용, 세금 합계 |
| 세후수익 | 계산값 | 예상 매도가에서 총 비용과 세금을 뺀 금액 |
| 수익률 | 계산값 | 세후수익을 필요 현금으로 나눈 비율 |

## 계산 방향

```txt
총 취득비용 =
입찰가
+ 취득세/지방세
+ 등기비용
+ 법무비
+ 미납관리비
+ 수리비
+ 명도비
+ 보유기간 이자
+ 중도상환수수료

필요 현금 =
총 취득비용 - 대출금액

세전수익 =
예상 매도가 - 입찰가

총 비용처리 금액 =
취득 관련 비용 + 보유비용 + 매도비용

과세표준 =
세전수익 - 비용처리 금액

세후수익 =
예상 매도가 - 총 비용 - 세금

수익률 =
세후수익 / 필요 현금
```

## 주의 문구

```txt
본 계산은 참고용입니다.
실제 대출 가능 여부는 금융기관 확인이 필요합니다.
세금은 세무 전문가 또는 관계 기관 확인이 필요합니다.
권리관계와 명도 가능성은 원본 서류와 현장 확인이 필요합니다.
```
