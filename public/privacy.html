개인정보처리방침

낙찰노트는 서비스 제공과 오류 개선을 위해 필요한 최소한의 정보를 처리합니다.

홈 서비스 소개 경매 가이드 개인정보처리방침 면책 고지 문의하기

1. 처리하는 정보

  • 사용자가 입력한 법원명, 사건연도, 사건번호 등 조회에 필요한 값
  • 서비스 이용 중 브라우저에 저장되는 저장 후보, 임시 후보, 메모, 입찰가 입력값
  • 서버 오류 확인을 위한 요청 시각, 요청 경로, requestId 등 기술 정보

2. 브라우저 저장소 사용

낙찰노트는 저장 후보와 일부 입력값을 사용자의 브라우저 localStorage 또는 sessionStorage에 저장할 수 있습니다. 공용 PC에서는 사용 후 저장 후보와 입력값을 초기화하는 것이 좋습니다.

3. 외부 API 사용

법원경매정보, 카카오 주소 검색, 국토교통부 실거래가, 온비드 등 외부 API 또는 공공 데이터 연동이 사용될 수 있습니다. API 키는 브라우저에 직접 노출하지 않고 서버에서 관리하는 것을 원칙으로 합니다.

4. 광고와 쿠키

향후 광고가 적용될 경우 광고 서비스 제공 과정에서 쿠키 또는 유사 기술이 사용될 수 있습니다. 광고 적용 시 관련 고지를 보완합니다.

5. 문의

개인정보 관련 문의는 문의하기 페이지를 통해 접수할 수 있습니다.
